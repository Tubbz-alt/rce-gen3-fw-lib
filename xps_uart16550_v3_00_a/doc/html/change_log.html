11.3 - Changes in VHDL/Verilog/Netlist sources (.vhd, .v, .ngc, .edn) 


 New Features:

Changes in v2.01.a, introduced in 11.1

11.1 - Changes in VHDL/Verilog/Netlist sources (.vhd, .v, .ngc, .edn) 


New Features:
  • Updated the helper libraries proc_common_v2_00_a to proc_common_v3_00_a and plbv46_slave_single_v1_00_a to plbv46_slave_single_v1_01_a
Resolved Issues:
  • Fixed <CR415474> (Break Interrupt Issue) :- Updated Receive state machine to detect the break error if sin input keeps low for full character and if framing error occurs before the break condition
    • Versions that have this issue: v2.00.a, v2.00.b
  • Core is updated to support +3% to -3% baud jitter in the receive data 
    • Versions that have this issue: v2.00.a, v2.00.b
  • Fixed <CR481176>  (Latch generation of lsr2_rst in uart16550.vhd)
    • Versions that have this issue: v2.00.a, v2.00.b
  • Fixed <CR490328> (Improved behavior of LSR(7): Error in receiver FIFO bit) Updated core to report error in the receiver FIFO until the last character containing error gets read out of the FIFO. 
    • Versions that have this issue: v2.00.a, v2.00.b
Known Issues / Limitations:
  • None

11.1 - Changes in documentation associated with core

  • Removed Virtex2p support.
